About Bug Tracker

Welcome to Bug Tracker, a simple yet effective application designed to help you manage and track software bugs efficiently.

This project is built using Spring Boot for the backend and Thymeleaf for the frontend, providing a robust and user-friendly experience.

Our Mission

Our mission is to provide a straightforward and intuitive platform for developers and testers to report, categorize, and resolve bugs, ensuring higher quality software releases.

Key Features

  • Easy bug reporting with detailed descriptions.
  • Categorization of bugs by priority and status.
  • User management to assign and track bugs.
  • Clean and responsive user interface.

Technologies Used

  • Backend: Spring Boot, Spring Security, Spring Data JPA, Hibernate
  • Database: H2 (in-memory for development)
  • Frontend: Thymeleaf, Bootstrap 5
  • Build Tool: Maven
